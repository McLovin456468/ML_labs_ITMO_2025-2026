## Лабораторная работа №1: Предобработка данных и прогнозирование стоимости квартир

### Цель

Построить модель прогнозирования стоимости квартиры (или цены за м²) на основе предобработанных данных.

### Задачи

| № | Задача |
|---|--------|
| 1 | Проверить соответствие типов данных |
| 2 | Обработать пропущенные значения |
| 3 | Удалить дубликаты |
| 4 | Обработать аномалии и выбросы |
| 5 | Закодировать категориальные признаки |
| 6 | Выполнить масштабирование / нормализацию |
| 7 | Добавить новые признаки (с обоснованием) |
| 8 | Оценить влияние признаков на целевую переменную, отобрать значимые |
| 9 | Выделить группы объектов с разным влиянием характеристик на цену |
| 10 | Построить и обучить модель прогнозирования цены / цены за м² / ценовой категории |

### Результаты

- Обоснование выбора методов предобработки
- Модель прогнозирования с оценкой качества
- Выводы по каждому этапу

## Лабораторная работа №2: Прогнозирование оттока клиентов банка

### Цель

Разработать классификатор для прогнозирования вероятности прекращения сотрудничества клиента с банком.

### Целевая метрика

- **F1-мера ≥ 0.59** на тестовой выборке
- ROC AUC (для дополнительного анализа)

### Данные

| Поле | Описание |
|------|----------|
| `Number` | индекс строки |
| `CustomerId` | уникальный идентификатор клиента |
| `Surname` | фамилия |
| `CreditScore` | кредитный рейтинг |
| `Geography` | страна |
| `Gender` | пол |
| `Age` | возраст |
| `Tenure` | лет в банке |
| `Balance` | баланс счёта |
| `NumOfProducts` | количество продуктов |
| `HasCrCard` | наличие кредитной карты |
| `IsActiveMember` | активность |
| `EstimatedSalary` | предполагаемая зарплата |
| `Exited` | факт ухода (целевой признак) |

### Задачи

| № | Задача |
|---|--------|
| 1 | EDA и предобработка |
| 2 | Обучение классификатора |
| 3 | Достижение F1 ≥ 0.59 |
| 4 | Анализ ROC AUC |
